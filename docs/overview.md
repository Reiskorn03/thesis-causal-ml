# Project Overview

This repository is a cleaned public showcase version of the code used in the bachelor thesis:

**Assessing the Impact of Organ Transplantation on Health Outcomes Using Machine Learning**

The repository is intended to document the structure of the thesis workflow, including:

- preprocessing of multi-table clinical data,
- modeling for causal inference,
- evaluation of causal survival forest results,
- metadata-driven feature handling.

The original raw data is not included in this repository.
Some scripts retain thesis specific path assumptions and are included for transparency of the original workflow rather than public reproducibility.
